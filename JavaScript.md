# 2.计算机基础

## 2.1计算机组成

- 硬件（输入设备、输出设备、CPU、硬盘(永久存储数据)、内存(暂时存储)）
- 软件（系统软件、应用软件）

## 2.2数据存储

- 计算机内部使用二进制0 和 1来表示数据
- 所有数据、程序（包括文件图片等都是以二进制数据的形式存放在硬盘里）

## 2.3数据存储单位

- bit < byte < kb < MB < GB <TB
- 1TB = 1024GB，1GB = 1024MB，1MB = 1024kB，1kB = 1024B，1B = 8b

## 2.4程序运行

- 打开某个程序时，先从硬盘(机械)中把程序的代码加载到内存(电)中
- CPU执行内存中的代码
- 硬盘--内存条--cpu
